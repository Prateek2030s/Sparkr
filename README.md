# Sparkr
Team ID 17 LifeHack
This is a web application built with **Next.js** and **Firebase** for features like authentication, analytics, and more.

---

## Requirements

- Node.js `>= 16`
- Firebase project (https://console.firebase.google.com/)
- npm
